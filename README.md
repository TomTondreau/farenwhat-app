# FahrenWhat?

Una aplicación web responsiva para la conversión de temperatura y una guía de cocción, diseñada con una estética inspirada en los hornos de cocina vintage.

## Características

*   **Conversor de Temperatura:** Convierte fácilmente entre grados Celsius y Fahrenheit.
*   **Guía de Cocción:** Proporciona temperaturas de cocción recomendadas para diversas carnes y puntos de cocción.
*   **Búsqueda de Guía:** Permite buscar cortes específicos dentro de la guía de cocción.
*   **Mensajes Dinámicos:** Un emoji y un texto informativo cambian dinámicamente según la temperatura ingresada, ofreciendo "facts" o estados relevantes.
*   **Diseño Responsivo:** Adaptada para verse y funcionar bien en dispositivos móviles, tablets y computadoras de escritorio.

## Diseño y Estilo

La aplicación adopta una paleta de colores y un estilo que evocan la calidez y el encanto de un horno de cocina clásico:

*   **Paleta de Colores:** Predominan los tonos crema para los fondos de las secciones, cafés oscuros y negros suaves para el fondo general y el texto, y acentos en verde-azul oscuro (`#274747`) para subtítulos y un naranja-marrón (`#bc6536`) para los títulos principales, inspirados en los colores de los hornos vintage.
*   **Logo:** Un logo grande y prominente que ocupa el ancho completo de su contenedor, diseñado para ser el punto focal visual y reflejar la temática de la aplicación.
*   **Tipografía:** Utiliza la fuente 'Inter' de Google Fonts para una legibilidad moderna y limpia.
*   **Inputs:** Campos de entrada con fondos blancos, texto oscuro y bordes sutiles para una integración armoniosa.
*   **Botones:** Botones con fondos marrones y texto crema claro para asegurar la legibilidad y mantener la coherencia estética.
*   **Animación de Fuego:** Un sutil y errático efecto de "fuego" o "brillo" en las secciones principales, con tonos naranjas y rojizos que simulan el parpadeo de una fogata, añadiendo un toque de ambiente.

## Tecnologías Utilizadas

*   **HTML5:** Estructura de la aplicación.
*   **Tailwind CSS:** Framework CSS para un diseño rápido y responsivo.
*   **JavaScript:** Lógica de conversión, manejo de la guía de cocción y actualización dinámica de la interfaz.

## Uso

1.  Clona el repositorio.
2.  Abre `index.html` en tu navegador web.
